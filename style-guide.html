Usage Guidelines

  • Primary: Main actions like "Upload Photos", "Save Changes"
  • Secondary: Alternative actions like "Cancel", "Go Back"
  • Ghost: Tertiary actions or navigation items
  • Danger: Destructive actions like "Delete", "Remove"

Implementation Notes

CSS Variables

All design tokens are defined as CSS custom properties for easy theming:

/* Example usage */ .my-element { color: var(--primary-blue); padding: var(--space-4); border-radius: var(--radius-md); }

Accessibility

  • All colors meet WCAG AA contrast requirements
  • Focus states are clearly visible with 3px outline
  • Interactive elements have minimum 44x44px touch targets
  • Semantic HTML elements used throughout

Animation Guidelines

  • Use 200ms duration for most transitions
  • Apply ease-in-out timing function
  • Subtle hover effects on interactive elements
  • Loading animations for async operations
